
The HL7eu and IHEeu imaging report group has finished processing the ballot comments of the [0.1.0 ballot](https://hl7.eu/fhir/imaging-r5/0.1.0-ballot/). Besides addressing these ballot comments we also added and R4 version of the specification. Both specifications are generated from a common source base and reflect the same content. The ballot candidate can be found [HL7 Europe Imaging Report R4 v1.0.0-ballot](https://build.fhir.org/ig/hl7-eu/imaging-r4/branches/ballot-0.2.0/en/).

In order to facilitate testing at the IHEeu Connecthaton in Brussels at the end of March (see [IHEeu XtEHR projecthaton](https://connectathon.ihe-europe.net/xt-ehr-projectathon-2026)), a snapshot has been generated: [HL7 Europe Imaging Report R4 v1.0.0-alpha](https://hl7.eu/fhir/imaging/xtehr/en/).

The current builds of the implementation guide can be found at the following URLs:

* [HL7 Europe Imaging Study Report R4](https://build.fhir.org/ig/hl7-eu/imaging-r4/en/)
* [HL7 Europe Imaging Study Report R5](https://build.fhir.org/ig/hl7-eu/imaging-r5/en/)
